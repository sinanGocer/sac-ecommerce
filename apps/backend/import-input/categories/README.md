# Kaydedilmiş Aveda kategori HTML dosyaları buraya

Tarayıcıdan kaydettiğiniz kategori/listeleme sayfalarını (`.html`) bu klasöre
koyun, sonra şunu çalıştırın:

```bash
cd apps/backend && npm run catalog:discovery:dry
```

Ayrıntılı talimat: `apps/backend/docs/category-discovery-guide.md`.

Not: Bu klasördeki `.html` dosyaları gitignore'dadır (repoya gönderilmez); yalnız
bu README izlenir.
